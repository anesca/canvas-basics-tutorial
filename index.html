HTML5 Canvas Basics

What is Canvas?

  • An element <canvas></canvas>
  • Provides a space to draw graphics on the fly using Javascript
  • If you were to inspect element on the HTML code you would not see anything inside of the canvas element as the graphics drawn inside of it are not part of the DOM
  • Can have many canvas elements on a page and they can overlap eachother
  • Canvas graphics cen be drawn in 2D or 3D
